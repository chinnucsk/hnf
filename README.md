hnf: hacker news filter
=======================

Usage
-----
        rebar get-deps
        rebar compile
        ./start-dev.sh

A `hnf` instance is usually running live at http://diff.biz/

Things to Change
----------------

* You can change the webserver's IP/Port in hnf.config.

Building
--------
Download deps:

        rebar get-deps

Build:

        rebar compile
